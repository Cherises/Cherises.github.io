Mac快捷键完全指南

刚切换到 Mac，面对全新的键盘布局和快捷键，确实需要一份详尽的指南。我把 MacBook Air 上最实用的快捷键分成了五大类，先从最常用的功能键说起。

一、顶部功能键（F1-F12）—— 最直观的快捷操作

MacBook Air 的顶部功能键默认直接控制系统硬件，不需要额外按 Fn 键：

按键 功能 说明
F1 降低屏幕亮度
F2 增加屏幕亮度
F3 调度中心 (Mission Control) 查看所有正在运行的窗口和桌面空间
F4 启动台 (Launchpad) 查看所有已安装的应用
F5 降低键盘背光亮度
F6 增加键盘背光亮度
F7 上一首/倒回 适用于音乐、视频
F8 播放/暂停
F9 下一首/快进
F10 静音
F11 减小音量
F12 增大音量

💡 小提示：如果想在特定应用中使用 F1-F12 作为标准功能键（比如编程或游戏），可以去「系统设置 > 键盘」勾选「将 F1、F2 等键用作标准功能键」，之后需要配合 Fn 键才能触发硬件功能。

二、截图与录屏 —— 最详细的完整指南

macOS 的截图功能非常强大，以下是完整的快捷键对照表：

快捷键 功能 保存位置
Command + Shift + 3 截取整个屏幕 桌面（PNG 文件）
Command + Shift + 4 截取选定区域（出现十字线后拖拽） 桌面
Command + Shift + 4 + 空格 截取特定窗口（光标变为相机图标后点击窗口） 桌面
Command + Shift + 5 打开截图/录屏控制中心（macOS Mojave 及以上） 可自定义

截图技巧进阶：

  • 截到剪贴板而非保存文件：在上述任意快捷键中加入 Control 键，截图会复制到剪贴板，可直接粘贴到其他应用

    • 例如：Command + Control + Shift + 4 = 截取区域到剪贴板

  • 去除窗口截图的阴影：用 Command + Shift + 4 + 空格 进入窗口截图模式后，按住 Option 键再点击，截出的窗口没有投影

  • 精确控制选区：在 Command + Shift + 4 的选区模式下：

    • 按住空格键可移动整个选区
    • 按住 Option 键可从中心点等比例缩放
    • 按 Esc 键取消截图

  • 带延时截图：通过 Command + Shift + 5 打开控制中心，可选择 5 秒或 10 秒延时

  • 录制屏幕：同样在 Command + Shift + 5 控制中心，可选择录制整个屏幕或选定区域

  • Touch Bar 截图（若机型配备）：Command + Shift + 6

三、通用编辑快捷键（与 Windows 对照）

Mac 的 Command 键（⌘）基本上对应 Windows 的 Ctrl 键，以下是常用对照：

操作 Mac 快捷键 Windows 对照
复制 Command + C Ctrl + C
粘贴 Command + V Ctrl + V
剪切 Command + X Ctrl + X
全选 Command + A Ctrl + A
撤销 Command + Z Ctrl + Z
重做 Command + Shift + Z Ctrl + Y
保存 Command + S Ctrl + S
查找 Command + F Ctrl + F
打印 Command + P Ctrl + P
关闭窗口 Command + W Ctrl + W
退出应用 Command + Q Ctrl + Q（完全不同）
打开新窗口 Command + N Ctrl + N

⚠️ 特别提醒：Mac 的「剪切」只能剪切文本内容。要「剪切」文件，正确操作是：

  1. Command + C 复制文件
  2. Command + Option + V 移动文件（相当于 Windows 的剪切粘贴）

四、窗口与系统管理

快捷键 功能
Command + Tab 切换应用（向前循环）
Command + Shift + Tab 切换应用（向后循环）
Command + `（Tab 上方的反引号） 在同一个应用的多个窗口间切换
Command + H 隐藏当前应用的所有窗口
Command + Option + H 隐藏除当前应用外的所有窗口
Command + M 最小化当前窗口至 Dock
Command + Option + M 最小化当前应用的所有窗口
Command + Option + Esc 强制退出应用（Mac 的任务管理器）
Control + 方向左/右 切换桌面空间
Control + 上方向 打开调度中心

关于 Fn 键的两种用法：

操作 功能
按一下 Fn（地球键） 切换输入法（可自定）
按两下 Fn 启动听写功能

五、启动组合键（开机时按住）

这些快捷键在 Mac 开机或重启时按住使用，非常有用来解决各种问题：

启动快捷键 功能
按住 Option (Alt) 进入启动磁盘管理器（选择从哪个系统启动）
按住 Shift 进入安全模式
按住 Command + R 进入恢复分区（重装系统、磁盘工具）
按住 T 进入目标磁盘模式（把 Mac 当移动硬盘用）
按住 Command + Option + P + R 重置 NVRAM（开机听到两次启动声后松手）
按住 D 运行 Apple Hardware Test 硬件检测
按住 Command + V Verbose 模式（显示详细启动日志）
按住 Command + S 单用户模式（命令行修复）

六、辅助功能快捷键

  • Command + Option + F5：打开辅助功能选项（VoiceOver、缩放等）
  • Control + Option + Command + 8：反转屏幕颜色

快捷键符号速查表

Mac 菜单中常出现的符号含义：

符号 按键
⌘ Command
⌃ Control
⌥ Option (Alt)
⇧ Shift
⇪ Caps Lock
fn 功能键（或地球键）

建议先记住 截图三件套（Cmd+Shift+3/4/5）和 常用编辑快捷键（Cmd+C/V/Z），其他的可以收藏这份指南，用到了再翻出来看。几天操作下来，这些快捷键就会变成肌肉记忆了！
