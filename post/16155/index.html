Ubuntu24系统软件优化推荐

Created2025-10-03|Updated2025-10-06|tech
|Word Count:1.2k|Reading Time:4mins|Post Views:

前情提要

我现在虽然安装了ubuntu24,但是仍然对kde的文件管理器很喜欢，对别的发行版本的一些常用工具都很青睐，反而ubuntu的gnome自带的一些系统应用很难用，所以我想集思广益，把各个开源linux分支的优秀软件拿过来给我的ubuntu用，比如昨天我用kde的fict5输入法就比ubuntu默认ibus的好用，然后kde的ksonle

现在是 Ubuntu 24 + GNOME 桌面，但“取各家之长”，把 KDE / XFCE / Deepin / Elementary / Mint 等发行版的优秀应用都搬过来用，让 Ubuntu 更顺手。

📂 文件管理器

  • GNOME 默认：Nautilus（功能单一，扩展少）

  • 更好替代：

    • Dolphin（KDE） → 最强文件管理器，支持分栏、标签、挂载、压缩、终端整合。
    • Thunar（XFCE） → 轻量快速，支持自定义右键动作。
    • Nemo（Cinnamon/Mint） → Nautilus 的超集，带有双面板、分栏、插件多。

📑 终端

  • GNOME 默认：Gnome Terminal

  • 更好替代：

    • Konsole（KDE） → 标签、分屏、配色方案强大。
    • Kitty → GPU 渲染，特效强大，支持连字、透明。
    • Alacritty → 极简 + 高性能。
    • Tilix（GNOME） → 平铺终端，多分屏好用。

🌐 浏览器

  • GNOME 默认：Web/Epiphany（太简陋）

  • 更好替代：

    • Firefox（官方推荐，插件生态强）
    • Chromium/Chrome
    • Falkon（KDE） → 基于 QtWebEngine，轻量。

🎶 音乐播放器

  • GNOME 默认：Rhythmbox

  • 更好替代：

    • Elisa（KDE） → 界面简洁，和 KDE 生态整合。
    • Clementine / Strawberry → 类似 iTunes，功能丰富。
    • Lollypop（GNOME 3rd） → 现代 UI，很美观。

🎥 视频播放器

  • GNOME 默认：Videos / Totem

  • 更好替代：

    • MPV → 极简 + 强大，命令行/快捷键控制。
    • SMPlayer（Qt） → MPV 前端，配置方便。
    • VLC → 万能播放器，UI 略笨重但功能全。

📸 截图 & 录屏

  • GNOME 默认：Screenshot / Screen Recorder（功能很少）

  • 更好替代：

    • KSnip（KDE） → 截图 + 标注。
    • Flameshot → 全平台强大截图工具。
    • OBS Studio → 专业录屏 + 推流。

📦 软件管理

  • GNOME 默认：GNOME Software

  • 更好替代：

    • Discover（KDE） → 包管理 + Flatpak + Snap 一体化。
    • Synaptic → 传统 apt 前端，最稳。
    • Nix Package Manager → 跨发行版通用。

📄 文本编辑器 / IDE

  • GNOME 默认：Gedit

  • 更好替代：

    • Kate（KDE） → 功能全面，插件多。
    • Geany（轻量 IDE） → 快速小巧。
    • VS Code / VSCodium → 强大编程环境。

📊 系统监控

  • GNOME 默认：System Monitor

  • 更好替代：

    • KSysGuard（KDE）
    • Bpytop / Bashtop（终端）
    • htop / btop（终端，更轻量）

💬 输入法

  • GNOME 默认：IBus

  • 更好替代：

    • Fcitx5（KDE 推荐） → 中文体验最佳，扩展性强。
    • Rime（可挂在 Fcitx5 / IBus 下）→ 极致定制。

📧 邮件客户端

  • GNOME 默认：Evolution（庞大，笨重）

  • 更好替代：

    • KMail（KDE）
    • Thunderbird（最通用）

📚 PDF / 文档阅读器

  • GNOME 默认：Evince

  • 更好替代：

    • Okular（KDE） → 支持 PDF / EPUB / Markdown，注释功能强大。
    • Zathura → 极简键盘操作。

✨ 这么一套下来，就能拥有：

  • Dolphin + Konsole + Fcitx5 → KDE 核心体验
  • Kitty/Alacritty + Flameshot + MPV + Okular → 跨桌面最优解
  • Synaptic + Thunderbird + Kate + Lollypop → 实用增强

🖼️ Snipaste 替代（截图 + 停靠功能）

Snipaste 最大的亮点是 截图后可贴在桌面上悬浮。Linux 下也有接近体验的工具：

  • KSnip → KDE 社区的截图工具，支持截图 + 标注，但没有悬浮。
  • Flameshot → 超好用截图 + 标注工具（Ubuntu 上最常见），但也不支持直接贴图。
  • Deepin Screenshot（deepin-screenshot） → 功能接近 Windows 的截图工具。
  • Peek → GIF 录屏神器。

👉 真正接近 Snipaste 悬浮贴图 的只有：

  • Paste（Snipaste for Linux 非官方移植） → 在 GitHub 上有项目，可以把截图贴到桌面上。
  • ScreenPen → 可以贴图 + 桌面绘图（不过比较小众）。

📦 BindZIP 替代（压缩/解压工具）

BindZIP 在 Windows 下是右键一键压缩/解压，Linux 下也有类似体验：

  • File Roller（GNOME 默认归档管理器） → 支持 zip/tar/7z，但功能简单。

  • Ark（KDE） → 功能更强大，支持 rar/7z/zip/tar 等各种格式，推荐。

  • PeaZip → 跨平台超强 GUI 压缩工具。

  • 命令行工具：

    • zip/unzip → 压缩/解压 zip
    • tar → 打包和压缩
    • 7z → 更高压缩率

📂 更好的文件浏览器

  • Dolphin（KDE） → 标签 + 分栏 + 预览 + 插件，功能最全。
  • Nemo（Cinnamon） → Nautilus 增强版，支持双面板 + 插件。
  • Thunar（XFCE） → 轻量，支持自定义右键动作。
  • Ranger（命令行） → 超强 CLI 文件浏览器，支持图片预览。
  • nnn（命令行） → 比 Ranger 还轻量，但功能依旧强大（快、插件多）。
  • Midnight Commander (mc) → 经典双栏命令行文件管理器（像 Norton Commander）。

🚀 建议的组合

如果你想要 像 Windows 一样图形化顺手：

  • 文件管理：Dolphin（全能）
  • 压缩：Ark + 7zip
  • 截图：KSnip / Flameshot + Paste（贴图工具）

如果你想要 极客风命令行体验：

  • 文件浏览：Ranger / nnn
  • 压缩：tar + 7z
  • 截图：Flameshot（命令行调用也行）
